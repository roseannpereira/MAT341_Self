Stage, Commit, and Push to GitHub
1. "git add <file name>"
2. "git commit -m "notes for yourself""
3. "git push"

Updating MAT341 GitHub Folder
1. Enter the MAT341 folder in the terminal
2. Type "git pull"

Cloning Repositories to Your Computer
1. Copy link to repository
2. Change directories to desktop or documents
3. Enter "git clone <link to repository>"
